# `%fund` Debugging

Enabling debug mode (and other configurations) in `%fund` is simple; just run
this command to enable it:

```
|cp /=fund=/cfg/xtra/verb/hoon /=fund=/cfg/verb/hoon
```

and this command to disable it:

```
|rm /=fund=/cfg/verb/hoon
```
